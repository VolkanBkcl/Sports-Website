Kilo Verme Programı

12 Haftalık Sağlıklı Kilo Verme Programı

Program Hakkında

Bu program, sağlıklı ve kalıcı kilo vermenizi sağlamak için özel olarak tasarlanmıştır. 12 haftalık süreçte, dengeli beslenme ve etkili egzersizlerle hedefinize ulaşacaksınız.

Haftalık Program

Pazartesi - Kardiyo ve Üst Vücut

  • 30 dakika koşu/bisiklet
  • Push-ups: 3 set x 12-15 tekrar
  • Dumbbell Rows: 3 set x 12-15 tekrar
  • Plank: 3 set x 45 saniye

Çarşamba - HIIT ve Core

  • 20 dakika HIIT antrenmanı
  • Mountain Climbers: 3 set x 30 saniye
  • Russian Twists: 3 set x 20 tekrar
  • Leg Raises: 3 set x 15 tekrar

Cuma - Kardiyo ve Alt Vücut

  • 30 dakika yürüyüş/koşu
  • Squats: 3 set x 15 tekrar
  • Lunges: 3 set x 12 tekrar
  • Glute Bridges: 3 set x 15 tekrar

Beslenme Rehberi

Günlük Kalori Hedefi

Vücut ağırlığınızın kg başına 25-30 kalori

Protein İhtiyacı

Vücut ağırlığınızın kg başına 1.6-1.8 gram

Örnek Öğün Planı

  • Kahvaltı: Yulaf ezmesi ve meyve
  • Ara Öğün: Elma ve badem
  • Öğle: Izgara tavuk salatası
  • Antrenman Sonrası: Protein shake
  • Akşam: Izgara balık ve sebze
