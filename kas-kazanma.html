Kas Kazanma Programı

8 Haftalık Özel Kas Geliştirme Programı

Program Hakkında

Bu program, kas kütlenizi artırmak ve güç kazanmak için özel olarak tasarlanmıştır. 8 haftalık süreçte, aşamalı olarak artan zorluk seviyeleriyle kaslarınızı maksimum seviyede geliştireceksiniz.

Haftalık Program

Pazartesi - Göğüs ve Triceps

  • Bench Press: 4 set x 8-10 tekrar
  • Incline Dumbbell Press: 3 set x 10-12 tekrar
  • Cable Flyes: 3 set x 12-15 tekrar
  • Triceps Pushdowns: 3 set x 12-15 tekrar

Çarşamba - Sırt ve Biceps

  • Pull-ups: 4 set x 8-10 tekrar
  • Bent Over Rows: 3 set x 10-12 tekrar
  • Lat Pulldowns: 3 set x 12-15 tekrar
  • Bicep Curls: 3 set x 12-15 tekrar

Cuma - Bacaklar

  • Squats: 4 set x 8-10 tekrar
  • Leg Press: 3 set x 10-12 tekrar
  • Romanian Deadlifts: 3 set x 10-12 tekrar
  • Calf Raises: 4 set x 15-20 tekrar

Beslenme Rehberi

Günlük Kalori Hedefi

Vücut ağırlığınızın kg başına 35-40 kalori

Protein İhtiyacı

Vücut ağırlığınızın kg başına 2-2.2 gram

Örnek Öğün Planı

  • Kahvaltı: Yulaf ezmesi, muz ve protein tozu
  • Ara Öğün: Protein shake ve fıstık ezmesi
  • Öğle: Izgara tavuk, pilav ve sebze
  • Antrenman Sonrası: Protein shake ve meyve
  • Akşam: Somon balığı, tatlı patates ve sebze
